# BMI Calculator using Python Tkinter

A simple and interactive BMI (Body Mass Index) Calculator desktop application built with Python and Tkinter.
This project allows users to enter their weight and height, calculate BMI instantly, and view their health category in a clean graphical user interface.

---

## Features

* Simple and user-friendly GUI
* Calculates BMI instantly
* Displays BMI with 2 decimal precision
* Shows BMI category:

  * Underweight
  * Normal
  * Overweight
  * Obese
* Error handling for invalid inputs
* Lightweight and beginner-friendly Python project

---

## Technologies Used

* Python
* Tkinter (Python GUI Library)

---


## How BMI is Calculated

BMI Formula:

BMI = \frac{weight\ (kg)}{height^2\ (m^2)}

Where:

* Weight is measured in kilograms (kg)
* Height is measured in meters (m)

---

## BMI Categories

| BMI Range      | Category    |
| -------------- | ----------- |
| Less than 18.5 | Underweight |
| 18.5 – 24.9    | Normal      |
| 25 – 29.9      | Overweight  |
| 30 and above   | Obese       |
